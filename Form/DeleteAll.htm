모두지우기

PC전용설정:미지원

설명

폼 또는 주어진 탭뷰 내의 모든 아톰들의 값을 지워준다.

매개변수

  1. 탭뷰번호 :값을 지워줄 뷰의 번호. 전체일 경우 폼 전체를 지운다. 
  2. 다시그리기 : 값을 지운다음 화면을 다시 그려줄지 여부(참, 거짓)
  3. 기본키삭제안함: 기본검색키가 설정된 아톰들의 값을 삭제 안함(참, 거짓)
  4. 전체 지우기: 모든 아톰의 값 지워줄지 여부(참, 거짓)

예문

  • 다음은 모두지우기 를 설정하는 예문이다.

   폼을 열면 다음 문단을 실행한다.
{
   폼(전체, 참, 참, 참)의 모두지우기를  실행한다 .
}
