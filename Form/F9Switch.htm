F9전환

PC전용설정:지원

설명

실행상태에서 F9 키를 눌렀을 때의 동작을 실행한다.

  • 선택표시 속성을 가지는 아톰들을 토글 시키는 기능을 수행한다. 선택표시속성이 체크된아톰들은 보이기/숨기기가 자동 토글된다.

예문

  • 다음은 F9전환를 실행하는 예문이다.

 

폼을 열면 다음문단을 실행한다 .
{
    입력란의 선택표시를 허용한다.
}

버튼1을 누르면 다음문단을 실행한다.
{
    선택표시를 전환한다.
}
