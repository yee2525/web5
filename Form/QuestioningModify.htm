질의수정

PC전용설정:미지원

설명

사용자가 정의한 DB처리문을 실시간으로 수정할 때 사용한다.

매개변수

  1. 사용자정의 쿼리문 : 사용자가 작성한 쿼리문.
  2. 구분: 수정할 부분을 나타내는 상수 '실행테이블', '실행필드', '입력변수', '실행조건', '정렬기준', '그룹기준', '결합문장' 
  3. 문자열 결합여부: 수정부분을 4번째 문자열로 치환할지 문자열 결합을 할 것인지 여부(참: 결합, 거짓: 치환)
  4. 수정할 구문

예문

  • 다음은 질의수정을 설정하는 예문이다.

     

폼을 열면 다음 문단을 실행한다.
{
    폼의 질의수정("Select * From 대출검색", '실행조건' , 참, "대출수량 > 3")을 실행한다 .
}
